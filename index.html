Projetos em Ciência de Dados

Desenvolvimento de um Painel Gerencial para mensurar a performance de uma empresa marketplace de restaurantes na India

Nesse projeto, utilizei habilidades de programação em Python e manipulação de dados para gerar insights valiosos e construir um dashboard interativo.
O objetivo foi acompanhar e analisar as principais métricas do negócio, fornecendo informações estratégicas para tomada de decisões.
Como resultado, desenvolvi um painel hospedado em um ambiente Cloud (Streamlit), proporcionando acesso fácil e conveniente a partir de qualquer dispositivo. O painel oferece uma interface intuitiva e visualmente atraente, permitindo que os usuários explorem os dados, visualizem gráficos dinâmicos e obtenham insights de forma rápida e eficiente.
Ao combinar programação, pensamento estratégico e lógica de negócios, fui capaz de entregar uma solução completa que auxilia na compreensão do desempenho do negócio e na identificação de oportunidades de melhoria.

Ensaio de Machine Learning

Este projeto focou na aplicação prática dos principais algoritmos de Machine Learning, abordando o treinamento, ajuste fino de hiperparâmetros e o equilíbrio entre overfitting e underfitting. Foram realizadas análises em tarefas de classificação, regressão e clusterização, proporcionando uma visão abrangente das nuances de cada técnica.

O produto final é composto por 7 tabelas apresentando o desempenho dos modelos em diferentes etapas do pipeline (treinamento, validação e teste), utilizando métricas chave para uma avaliação completa de cada algoritmo.

As ferramentas utilizadas foram:

  • Python 3.9.12, Pandas 2.0.1, Numpy 1.23.5 e Scikit-learn 1.5.1.
  • Algoritmos de classificação: KNN, Decision Tree, Random Forest e Logistic Regression.
  • Métricas de performance: Accuracy, Precision, Recall, F1-Score, R2, MSE, RMSE, MAE, MAPE e Silhouette Score.
  • Algoritmos de regressão: Linear Regression Lasso, Linear Regression Ridge, Linear Regression Elastic Net, Decision Tree Regressor, Random Forest Regressor e Polynomial Regression.
  • Algoritmos de clusterização: K-Means e Affinity Propagation.
